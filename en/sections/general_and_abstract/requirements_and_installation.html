General & Abstract » Requirements & Installation

PHP requirements

  • PHP 5.1 (or newer)

    leaf framework is designed to work only to these versions, because some features that in previous versions required extra installation/configuration steps, are now bundled and enabled by default.

  • Reflection extension (pure optional)

    leaf frameowork will use Reflection to figure out dynamically how your classes/methods are declared.

  • XDebug (pure optional)

    It is recommended when developing PHP applications to use XDebug.

Apache requirements

  • Any version of Apache webserver

    We just luv Apache. Soon to be tested under lighttpd...

  • mod_rewrite enabled

    mod_rewrite (for the time being...) is leaf`s backbone :-).

  • support for .htaccess configuration files

    Required to setup the mod_rewrite, if not able to access directly Apache`s configuration files.
