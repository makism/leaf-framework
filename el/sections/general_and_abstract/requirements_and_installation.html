Απαιτήσεις της PHP

  • PHP 5.1 (ή νεότερη)

    Το leaf framework είναι σχεδιασμένο να λειτουργεί με παραπάνω εκδόσεις, επειδή μερικά χαρακτηριστικά τα οποία στις προγενέστερες εκδόσεις χρειάζονταν επιπλέον εγκαταστάσεις και ρυθμίσεις, είναι πλέον εξ ορισμού ενεργοποιημένα.

  • Reflection extension (καθαρά προαιρετικό)

    Το leaf frameowork θα χρησιμοποιήσει Reflection ώστε να ανακαλύψει δυναμικά πως είναι οι κλάσεις και οι μέθοδοι σας έχουν δηλωθεί.

  • XDebug (καθαρά προαιρετικό)

    Προτείνετε η χρήση του XDebug όταν αναπτύσσετε εφαρμογές PHP.

Απαιτήσεις του Apache

  • Οποιαδήποτε έκδοση του εξυπηρετητή Apache απλά κάνει...

    Αγαπάμε τον Apache. Σύντομα θα δοκιμάσουμε και τον lighttpd...

  • Ενεργοποιημένο mod_rewrite

    mod_rewrite (για την ώρα...) αποτελεί την ραχοκοκαλιά του leaf framework :-). Αυτό γιατί, το mod_rewrite χρησιμοποιείται, για να ξεχωρίσει τον Controller και τα Actions που εκτελούνται, καθώς επίσης και τα επιπλέον στοιχεία που μπορούν να υπάρξουν στο Url.

  • Υποστήριξη για αρχεία παραμέτρων .htaccess

    Επιβάλετε ώστε να ρυθμίσουμε τον mod_rewrite, εκτός και αν έχετε απευθείας πρόσβαση στα αρχεία παραμέτρων του Apache.

Εγκατάσταση

  1. Αποκτήστε ένα αντίγραφο του leaf framework

    Για την ώρα δεν υπάρχει κάποια επίσημη έκδοση, οπότε, ανατρέξτε στο http://leaf.teicrete.gr και ακολουθείστε τις οδηγίες που σας παρέχονται ώστε κάνετε checkout τον κώδικα στον κατάλογο /home/leaf/public_html.

  2. Ρύθμιση .htaccess

    Άς υποθέσουμε ότι είστε ο χρήστης "leaf" στο localhost και η προσωπική σας σελίδα είναι "http://localhost/~leaf".
    Τροποποιούμε το κρυφό αρχείο .htaccess ώστε η γραμμή που περιέχει το

    RewriteBase /
    να γίνει
    RewriteBase /~leaf/

  3. Ρύθμιση general.php

    Στο αρχείο leaf/etc/general.php, θα πρέπει να το αλλάξουμε ώστε οι μεταβλητές να γίνουν:

    $general['hostname'] = "http://localhost"
    και
    $general['base_dir' = "/~leaf/"
