Class leaf_Router

Description
Description | Vars (details) | Methods (details) Constants (details)

Processes and filters the current Uri.

Filters the Uri for erroneous characters, splits the query string (if enabled), matches the virtual file extension (if enabled), discovers the requested Controller(class) and the Action(method).

array $segments = NULL (line 72)

The extra segments found in the Uri.

Whatever follows after the method name, and is separated by "/", is considered to be a segment.
Example:
http://localhost/Blog/view/2007/xx/xx/aTitle/
The array $segments, will be populated by the strings:
"2007", "xx", "xx", "aTitle"
