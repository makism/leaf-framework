1.0 release

Από την στιγμή που γενικά δουλεύει όπως πρέπει το μεγαλύτερο ποσοστό του framework, έχουνε μείνει μερικά θέματα που θα πρέπει να διευθετηθούν. Έτσι, η πρώτη επίσημη έκδοση θα πρέπει να περιέχει (σε σταθερή κατάσταση) τα παρακάτω:

  • Request class

  • Response class

  • Loader class

  • Exception class

  • Api Docs

    θα δημιουργηθούν τελευταία ή θα ανανεωθούν αν χρειαστεί με το PhpDoc.

  • Εγχειρίδιο

    αυτή τη στιγμή συγγράφεται.

  • Λογότυπα
