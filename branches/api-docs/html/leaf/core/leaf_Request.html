Class leaf_Request

Description
Description | Vars (details) | Methods (details) Constants (details)

Provides access to all elements that compose the Uri.

This means, that we have request and refer to the file that the requested Controller is located in, the Action, the extra segments and finally the query string.
Also, this class implements some basic methods related with the Uri handling, like redirecting and Uri-reconstruction.
