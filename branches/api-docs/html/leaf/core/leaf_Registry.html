leaf_Framework Registry Class

Αυτή η κλάση υλοποιεί το πρότυπο σχέδιο Registry. Με αυτό το τρόπο, όλα τα αντικείμενα αποθηκεύονται σχεσιασιακά μέσα στην κλάση αυτή, και αναφερόμαστε μέσω αυτής.
